Date: 2026-06-04

## What the app does today

- Loads 3 training sessions (75 exercises, 43 session items) from Supabase
- Displays session list with title, description, difficulty, item count
- Tapping a session opens the player and auto-plays audio from Supabase storage
- Player: sequential playback, repetition tracking, next/prev, track list tap to jump
- Hive cache: all 3 tables cached after remote fetch; offline fallback works
- User can edit a session: opens with real exercises, adjust reps, reorder, save
  - Editing a server session creates a new user-owned copy (never mutates server data)
  - Editing a user session updates it in-place
  - User-created sessions appear in list alongside server sessions ("Yours" label)
  - User sessions survive remote refresh (upsert sync, not full wipe)
- Download sessions for offline audio playback (Dio, file system)
